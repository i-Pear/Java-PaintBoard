#### A painting board designed for Java homework.

#### Supports:

- Select Tool
- Pen Tool
- Line Tool
- Rectangle Tool
- Circle Tool
- Ellipse Tool
- Text Tool
- Insert pictures

- Fill options
- Font options
- Color options
- Line Type options
- Line Width options

- Undo / Redo Operations

- Multi-tabs Working

- Layer Rasterize
- Layer Hierarchical Relationship Adjustment

#### Based on JavaFX.
